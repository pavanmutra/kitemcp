# Dashboard Screenshots

Add your screenshots here.

## How to capture masked screenshots:

1. Run the dashboard: `npm run web`
2. Open http://localhost:3000
3. Click the eye icon (👁️) in the top-right corner to mask P&L values
4. Use your screenshot tool
5. Save to this folder with names:
   - `dashboard.png`
   - `deep-value.png`
   - `holdings.png`

Recommended: Use snipping tool or PowerToys on Windows.
